We've developed a lightweight desktop application called QuickTechDataSyncService that serves as a centralized data synchronization hub for your POS system. Built with WPF and .NET 8, this service creates a web API endpoint that mobile devices can connect to for retrieving and updating data. The application features a clean user interface with controls for starting/stopping the server and monitoring connection status, along with a comprehensive logging system to track synchronization activities. It connects to your existing SQL Server database and exposes endpoints for products, categories, customers, suppliers, transactions, and business settings. The service includes pagination support, error handling, and detailed sync logs to ensure reliable data transfer between your main POS system and mobile clients.
